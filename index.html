Academic Projects

Medical Chatbot with LLaMA & RAG

For my MSc dissertation, I developed an intelligent chatbot designed to provide accurate responses from medical datasets. The project workflow included:

  • Pretrained Model Utilization: Leveraged the LLaMA pretrained language model as a foundation.
  • Fine-Tuning with PEFT: Applied Parameter-Efficient Fine-Tuning (PEFT) to adapt the model to domain-specific medical data.
  • Retrieval-Augmented Generation (RAG): Implemented RAG to enhance response accuracy and validate results against the fine-tuned model.
  • Result Analysis: Conducted rigorous evaluation to identify hallucinations and compare performance across different approaches.

This project demonstrates hands-on experience with cutting-edge NLP techniques, including fine-tuning large language models, knowledge retrieval, and critical evaluation of AI outputs in a high-stakes domain.

Big Data Pipeline for Nutritional Data Analysis

As an independent project, I developed a big data pipeline to analyze large-scale nutritional datasets, exploring patterns, trends, and insights at scale. The project emphasized end-to-end data processing, distributed computing, and interactive analytics:

  • End-to-End Pipeline Development: Built a scalable workflow to ingest, process, and analyze multi-GB nutritional datasets from Kaggle.
  • Distributed Infrastructure: Deployed a Dockerized Hadoop cluster and used Ambari for service configuration, HDFS monitoring, and managing distributed storage.
  • Cluster Administration: Managed remote nodes via SSH (Putty), handling debugging, command-line operations, and workflow orchestration across the cluster.
  • ETL Workflow Design: Executed full ETL processes, transforming raw datasets into structured, analytics-ready formats in HDFS.
  • Large-Scale Analytics: Performed distributed queries and aggregations using Apache Hive, mining nutrient trends and category-level insights.
  • Interactive Visualization: Built dynamic dashboards in Apache Zeppelin to visualize nutrient patterns, temporal trends, and actionable insights.
  • Optimization and Scalability: Tuned cluster performance and optimized queries to ensure reproducibility and efficiency at scale.

AI-Driven Cardiovascular Risk Modeling

This project involved designing an end-to-end cardiovascular risk prediction pipeline using both classical Machine Learning and Deep Learning techniques. The aim was to build robust models capable of accurately predicting risk from clinical datasets, even when data is incomplete or imbalanced.

  • Machine Learning & Deep Learning: Developed models including Random Forest, Logistic Regression, and Artificial Neural Networks (ANN), achieving up to 88% prediction accuracy.
  • Data Preprocessing & EDA: Cleaned and analyzed clinical datasets, handled missing values with LGBMRegressor (numerical) and LGBMClassifier (categorical), applied Quantile Transformation for normalization, and mitigated outliers to improve model robustness.
  • Feature Engineering: Designed research-driven features to enhance predictive performance and improve model interpretability.
  • Hyperparameter Optimization: Used GridSearchCV and RandomizedSearchCV for tuning ML models and ANN architectures (layers, neurons, optimizers).
  • Handling Imbalanced Data: Applied SMOTE to improve generalization for minority classes.
  • Model Evaluation: Conducted cross-validation, ROC-AUC analysis, and evaluated performance using Confusion Matrices, Precision, Recall, and F1-score to ensure robust predictions.

Cloud-Native Media Distribution Platform

This project involved designing and implementing a scalable, cloud-native web application for media distribution, conceptually similar to platforms like Instagram, with a focus on video content sharing. The primary aim was to create a platform that supports distinct user roles while ensuring scalability, performance, and cloud-native best practices.

  • Creator & Consumer User Roles: Developed dedicated views for "creator" accounts to upload videos with metadata (Title, Caption, Location, Tags) and for "consumer" accounts to search, view, comment, and rate content without upload privileges.
  • Cloud-Native Architecture: Deployed the application on Azure, leveraging static web hosting for the frontend and REST endpoints for backend service logic and storage integration.
  • Data Persistence: Implemented scalable persistence using hosted databases and Azure Blob Storage to handle user data and media content efficiently.
  • Authentication & Access Control: Incorporated role-based authentication and user access control to secure the platform and manage different user capabilities.
  • Media Processing & Cognitive Services: Integrated video processing services and explored cognitive services for metadata extraction and media enhancements.
  • Deployment & Testing: Fully deployed the application on Azure and conducted thorough testing to validate functionality, performance, and scalability.

This project demonstrates expertise in cloud-native application development, scalable system design, and delivering role-based interactive media platforms in a real-world cloud environment.

Intelligent Movie Recommendation engine

During my Bachelor's final year project, I designed a research-focused movie recommendation system aimed at exploring the intersection of user behavior and content features. The system combined multiple recommendation strategies to provide more meaningful and personalized suggestions:

  • Content-Based Analysis: Investigated movie features such as genre, cast, and storyline to recommend films aligned with user preferences.
  • Popularity Insights: Studied trending movies to understand collective audience preferences and integrate them into recommendations.
  • User Feedback Adaptation: Incorporated likes and dislikes to dynamically refine recommendations, emphasizing the personalization aspect.
  • Genre-Focused Recommendations: Explored genre-specific patterns to guide users toward films that matched their thematic interests.

This project combined data analysis, algorithmic experimentation, and user-centered design to create a recommendation system that not only suggests movies but also uncovers patterns in viewing behavior.
